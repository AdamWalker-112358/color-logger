Color Logger

Only supported in Google Chrome and Chromium based browsers



Instructions


1. Open the console and type 
logger.colorName(styledText, styles [,unstyled text])

2. For example, to print a blue string to the console type: 
logger.blue('adam')
3. You can optionally pass and array of strings that will be appended without styles.

4. You can also pass your own style string to override the built in styles.
